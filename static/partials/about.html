What is Graphical User Interface for DNA Editing Screens (GUIDES)?

The development of CRISPR (clustered regularly interspaced short palindromic repeat)-associated Cas9 nuclease has enabled new methods for interrogating gene function. Given the simple programmability of CRISPR nucleases through short RNA guides, genome-scale knockout libraries have emerged as popular tools for phenotypic screens [1]. Other screening applications may require libraries that target particular sets of genes, have deeper coverage with more guide RNAs per gene, or target specific cell types or tissues. GUIDES (Graphical User Interface for DNA Editing Screens) is a web application to design customized CRISPR knockout libraries as easily as possible without sacrificing control.

How does it work?

Technical Specifications

GUIDES provides a complete pipeline for the design of custom single-guide RNA (sgRNA) libraries for use in a wide variety of applications. Briefly, GUIDES first finds sgRNAs within the consensus coding sequences of a user-specific list of genes. To prioritize sgRNAs for each gene, it integrates tissue-specific exon expression from the GTEx Consortium [2], on-target scores for higher activity (Doench et al. and Microsoft Azimuth) and also off-target avoidance [3-4] to optimize guide selection. When possible, GUIDES also targets functional protein domains based on Pfam data [6], as mutagenesis of functional protein domains leads to a higher proportion of null mutations and enhanced severity of negative selection [6].
